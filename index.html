This checklist is filled to the best of my ability based on my own playthrough, and should be mostly complete. However, will probably be things I may have missed due to new patches, DLC content, and such. On top of that, there seems to be various minor Merc Missions that have very fickle or specific trigger requirements that can be easily missed in any given playthrough. If you've discovered anything missing and would like see it added to this list, feel free to create a GitHub issue or PM me on Reddit (u/r52) with the data for the listing and I'll add it as soon as possible.
